LifeCity V1.6.8 — Roofs / Labels / Dev Editor

Изменения:
1. Удалены визуальные подложки buildingZones с названиями кварталов.
   - drawBuildingZones() теперь отключен.
   - данные buildingZones оставлены только для совместимости старых правок.

2. Убраны маленькие дубли названий зданий на крышах.
   - drawRoofLabel() отключен.
   - основные названия зданий остаются на уникальных 2.5D-вывесках/фасадах.

3. Переработаны крыши зданий в более выраженный 2.5D-формат.
   - добавлены свесы, парапеты, ребра, техблоки, skylight-блоки, вентиляция и тени.
   - разные типы крыш для жилого дома, банка, мэрии, полиции, магазина, кафе, аптеки, автосервиса, мусорки и завода.

4. Dev Mode теперь умеет двигать фонари и таблички.
   - фонари и street signs добавлены в editable registry.
   - перетаскивание сохраняется на сервер через devUpdateMapObject.
   - сохраненные координаты применяются к roadside layout.
   - кнопка “Записать карту в основной код” переносит правки фонарей/табличек в streetLightOverrides и streetSignOverrides.

5. Сервер обновлен.
   - добавлены buckets streetLights и streetSigns в data/map-objects-db.json.
   - добавлена sanitize-логика для roadside-объектов.
   - добавлен commit правок в public/js/city-layout.js.

Проверки:
- npm run check: OK
- browser smoke: OK
