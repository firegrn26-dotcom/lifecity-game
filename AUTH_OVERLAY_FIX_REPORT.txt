LifeCity Auth Overlay Fix
=========================

Проблема:
- После восстановления окна авторизации полноэкранный DOM-auth слой мог оставаться поверх canvas.
- Из-за этого после авторизации карта/игра могла быть перекрыта невидимым или полупрозрачным фоном.

Что исправлено:
- public/js/ui-auth-visibility-fix.js полностью переписан.
- После успешной авторизации #lc-auth-root теперь удаляется из DOM, а не просто скрывается.
- Старые auth-слои и legacy-кнопки принудительно скрываются.
- Добавлена защита body.lc-auth-passed, чтобы auth-слой не мог перекрыть canvas.
- В render-loop.js исправлено pointerEvents для активного окна авторизации.
- Проверка npm run check проходит успешно.
