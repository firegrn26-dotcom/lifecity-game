LifeCity 2.5D — Dev Map Commit To Source

Добавлено:
- Кнопка в меню разработчика: "Записать карту в основной код".
- Серверный обработчик Socket.IO: devCommitMapObjectsToSource.
- Перенос сохраненных правок из data/map-objects-db.json в основные файлы карты:
  - public/js/map.js — buildings + recyclingFactory const;
  - public/js/city-layout.js — roads, buildingZones, parks.
- Перед записью сервер создает backup-файлы рядом с исходниками:
  - map.js.bak-<timestamp>
  - city-layout.js.bak-<timestamp>
- После успешной записи временный data/map-objects-db.json очищается, потому координаты становятся основными.

Как пользоваться:
1. Включить Dev Mode.
2. Включить "Редактировать объекты карты".
3. Передвинуть/изменить объект, отпустить мышь — правка сохранится во временный JSON.
4. Нажать "Записать карту в основной код".
5. Сервер перенесет все временные правки в map.js / city-layout.js.
6. После этого нужно закоммитить измененные файлы в GitHub, если проект деплоится на Render.

Важно:
- На локальном сервере запись в код работает напрямую.
- На Render изменения файлов могут пропасть после redeploy, если не перенести их в GitHub.

Проверка:
- npm run check проходит успешно.
