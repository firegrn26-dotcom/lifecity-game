LifeCity Auth Buttons + Remember Checkboxes Fix

Изменено:
- Обе кнопки авторизации теперь используют один класс lc-auth-btn и имеют одинаковый стиль/размер.
- Левая кнопка всегда отвечает за вход:
  - режим входа: "Войти" отправляет логин;
  - режим регистрации: "Вход" возвращает в форму входа.
- Правая кнопка всегда отвечает за регистрацию:
  - режим входа: "Регистрация" открывает форму регистрации;
  - режим регистрации: "Зарегистрироваться" отправляет регистрацию.
- Возвращены чекбоксы:
  - Запомнить логин;
  - Запомнить пароль;
  - Показать пароль.
- Поля и карточка регистрации поджаты по высоте, чтобы кнопки стояли ровно друг напротив друга.
- Сохранение логина/пароля синхронизировано с localStorage и старыми ключами lifecity_remember_login / lifecity_remember_password.
