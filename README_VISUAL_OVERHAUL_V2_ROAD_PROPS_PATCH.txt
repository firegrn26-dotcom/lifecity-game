LifeCity — Visual Overhaul V2 Road Props Patch

Исправления:
- Фонари теперь ставятся по краям дорог, с учетом реальной ширины дороги и тротуарного отступа.
- Увеличена дистанция между фонарями, чтобы они не стояли слишком близко друг к другу.
- Добавлен глобальный minDistance между объектами, чтобы фонари не слипались на перекрестках.
- Фонари больше не ставятся прямо в зоне перекрестков.
- Таблички теперь тоже привязаны к краю дороги, а не смещаются случайно вниз.
- Таблички вынесены слегка наружу от дороги и имеют отдельную стойку.

Запуск:
1. npm install
2. npm start
