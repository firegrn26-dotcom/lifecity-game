LifeCity — Visual Overhaul V2

Что добавлено:
- Освещение 2.0: фонари вдоль дорог, мягкий glow, световые круги.
- Живой ночной город: светящиеся окна зданий, cinematic tint, виньетка.
- Атмосфера: дождь V1, частицы/пыль в воздухе, легкий экранный scanline-эффект.
- Детализация дорог: микротрещины/следы на полотне дороги.
- City props: небольшие городские таблички рядом с дорогами.
- UI polish: бейдж сборки обновлен на Visual V2, музыкальная панель получила более премиальный стиль.

Проверка:
- public/game.js прошел node --check
- local-server.js прошел node --check

Запуск:
1. Открой PowerShell в папке проекта.
2. Выполни npm install, если зависимости еще не установлены.
3. Выполни npm start
4. Открой http://localhost:3000

Важно:
V2 сделан без изменения серверной логики, размеров зданий, коллизий и механик.
