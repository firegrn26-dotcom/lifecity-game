LifeCity — исправленная версия

Что изменено:
1. Исправлен запуск проекта через npm start.
2. package.json теперь указывает main: local-server.js и содержит скрипт проверки npm run check.
3. Серверная раздача public теперь привязана к папке проекта, а не к текущей папке PowerShell.
4. Серверные зоны Мусорки и Магазина синхронизированы с клиентскими размерами зданий.
   Раньше клиент мог показывать действие, но сервер отклонял его из-за меньшей зоны.
5. Удалена улица Сервисная из клиента и из серверных зон спавна мусора.
6. В UI для недоступных действий теперь явно написано: НЕТ ДОСТУПНЫХ ДЕЙСТВИЙ и причина.
7. Проверена синтаксическая валидность local-server.js и public/game.js.
8. Архив очищен от node_modules, .git, Beckup и test.txt.

Запуск:
1. Открой PowerShell в папке проекта.
2. Выполни: npm install
3. Выполни: npm start
4. Открой: http://localhost:3000

Проверка кода:
npm run check
