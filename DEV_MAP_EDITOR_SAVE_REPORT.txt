LifeCity — Dev Map Editor Save Update

Версия: V1_6_6_dev_map_editor_save

Добавлено:
- Новый клиентский модуль public/js/dev-map-editor.js.
- В меню разработчика пункт "Редактировать объекты карты" теперь работает не только как временное движение зданий.
- Можно двигать:
  - здания;
  - парки / декоративные зоны;
  - подложки кварталов / buildingZones;
  - дороги целиком.
- Shift при перетаскивании прямоугольных объектов меняет ширину.
- Ctrl при перетаскивании прямоугольных объектов меняет высоту.
- После отпускания мыши объект автоматически отправляется на сервер.
- Сервер сохраняет изменения в data/map-objects-db.json.
- Сервер рассылает обновление всем подключенным клиентам.
- Клиент при загрузке повторно запрашивает сохраненное состояние карты.

Сервер:
- local-server.js получил отдельную базу карты: data/map-objects-db.json.
- Добавлены события Socket.IO:
  - devUpdateMapObject
  - requestMapObjectsState
  - mapObjectsState
  - mapObjectUpdated

Важно:
- Серверная логика игроков, работ, мусора и авторизации не менялась.
- ensureRecyclingFactoryOnMap больше не сбрасывает координаты завода, если он был перемещен через dev mode.

Проверки:
- npm run check — OK.
- local-server.js — JS syntax OK.
